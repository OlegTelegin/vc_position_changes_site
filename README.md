This repo hosts a static site from `docs/`.

### Configuration

1. In GitHub, go to Settings â†’ Pages.

### Custom domain

The `docs/CNAME` file sets the domain to `revelio_positions.olegteleg.in`.
Update your DNS with a `CNAME` record that points
`revelio_positions.olegteleg.in` to your GitHub Pages domain.
